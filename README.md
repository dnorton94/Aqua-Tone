# OpenAI-Build-Week-2026

##Hardware Requirements
Elegoo Uno R3 (Arduino Uno compatible)
USB Type-A to USB Type-B cable (the square "printer" USB cable) from the Uno to your computer
PCA9685 16-channel servo driver
MF90 servo
USB-C PD power board
USB-C wall charger and USB-C cable
Jumper wires
Breadboard
